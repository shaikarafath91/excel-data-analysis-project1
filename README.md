# Excel Data Analysis Project

## Project Overview
This project focuses on transforming raw electronic store sales data into an interactive and visually appealing Excel dashboard to support better business decision-making.

## Objective
- Analyze sales performance across products, regions, and time periods
- Identify key business trends and insights
- Present data in a clear and interactive dashboard format

## Dataset
- Raw sales data containing product details, sales quantity, revenue, region, and date
- Data was cleaned and structured for analysis using Microsoft Excel

## Tools & Technologies Used
- Microsoft Excel
- PivotTables and PivotCharts
- Advanced Excel formulas (XLOOKUP, IF, SUMIFS)
- Data cleaning and formatting techniques
- Excel dashboards and visualizations

## Key Tasks Performed
- Cleaned raw sales data by removing duplicates and handling missing values
- Created calculated columns using advanced Excel formulas
- Built PivotTables to analyze sales by product, category, and region
- Designed interactive dashboards with charts and slicers
- Automated repetitive calculations to improve efficiency

## Key Insights
- Identified top-performing products and regions
- Analyzed monthly and yearly sales trends
- Highlighted low-performing areas for business improvement
